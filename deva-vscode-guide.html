✦ Step 3 — Add Firebase Config to Your Code
1
Open index.html and Find the Config Section
2 Minutes
▼
In VS Code, click on index.html in the Explorer panel. Press Ctrl+F to open Find, and search for:
YOUR_API_KEY
You will see this block at the top of the file — replace ALL the placeholder values with your real Firebase config:
// BEFORE (placeholder values — won't work): const firebaseConfig = { apiKey: "YOUR_API_KEY", authDomain: "YOUR_PROJECT.firebaseapp.com", projectId: "YOUR_PROJECT_ID", storageBucket: "YOUR_PROJECT.appspot.com", messagingSenderId: "YOUR_SENDER_ID", appId: "YOUR_APP_ID" }; // AFTER (your real values from Firebase Console): const firebaseConfig = { apiKey: "AIzaSyD-XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX", authDomain: "deva-ai-12345.firebaseapp.com", projectId: "deva-ai-12345", storageBucket: "deva-ai-12345.appspot.com", messagingSenderId: "123456789012", appId: "1:123456789012:web:abcdef1234567890" };
💡Get these values from: Firebase Console → Project Settings (⚙️) → Your Apps → Web App → SDK setup and configuration
2
Add Your Admin Email
1 Minute
▼
In index.html, press Ctrl+F and search for admin@devaai.com — replace it with your own email:
// Find this line in index.html: isAdmin: user.email === 'admin@devaai.com', // Change to YOUR email: isAdmin: user.email === 'youremail@gmail.com',

Add Razorpay Key
1 Minute
▼
In index.html, find YOUR_RAZORPAY_KEY and replace it:
// Find this line: const RAZORPAY_KEY = 'YOUR_RAZORPAY_KEY'; // Replace with your actual key from razorpay.com: const RAZORPAY_KEY = 'rzp_live_XXXXXXXXXXXXXXXXX';
⚠️Use rzp_test_ key while testing, switch to rzp_live_ when going live
4
Add Anthropic API Key to App
1 Minute
▼
Open app.html in VS Code. The API key is entered by users through the sidebar UI — but you can set a default for testing. In the app, the user pastes their Claude API key in the sidebar and it's saved to localStorage.
📌For a production app, you should move the API key to a backend server so users don't need to provide it. But for MVP/launch, the current system works fine.
✦ Step 4 — Run & Test Locally
1
Start Live Server
30 Seconds
▼
With index.html open in VS Code, look at the bottom status bar — you'll see "Go Live" button. Click it!
Your browser will open at http://127.0.0.1:5500/index.html — this is your app running locally!
index.html — deva-ai
Press Alt+L Alt+O or click Go Live in status bar
index.html 🟢 Port: 5500
✅Every time you save a file (Ctrl+S), the browser auto-refreshes! No need to manually reload.
⚠️Firebase Google Login won't work on localhost — you need to add localhost to Firebase authorized domains. Go to: Firebase Console → Authentication → Settings → Authorized domains → Add localhost

✦ Step 5 — Push to GitHub
1
Create GitHub Repository
3 Minutes
▼
1. Go to github.com → Sign up / Log in
2. Click the + icon (top right) → New repository
3. Name: deva-ai
4. Select: Private (so your API keys are safe)
5. Click Create repository
⚠️Never put your API keys directly in the code for a PUBLIC repository — always use Private repos or environment variables
2
Push Your Code from VS Code
3 Minutes
▼
Open VS Code Terminal (Ctrl+`) and run these commands one by one:
$ git init
Initialized empty Git repository in /deva-ai/.git/
$ git add .
(adds all files)
$ git commit -m "Initial Deva AI launch"
[main (root-commit) abc1234] Initial Deva AI launch
4 files changed, 2847 insertions(+)
$ git remote add origin https://github.com/YOUR_USERNAME/deva-ai.git
$ git push -u origin main
✓ Branch 'main' set up to track remote branch 'main' from 'origin'.
💡Replace YOUR_USERNAME with your actual GitHub username. The URL is shown on your GitHub repository page after creating it.
For future changes — whenever you edit files, run:
$ git add .
$ git commit -m "Updated chat feature"
$ git push
✓ Vercel auto-deploys in ~30 seconds!
✦ Step 6 — Deploy to Vercel (Go Live!)
1
Connect GitHub to Vercel
5 Minutes
▼
Go to vercel.com → Click "Sign up" → Choose "Continue with GitHub"
Click "Add New Project" → Select "Import Git Repository"
Find deva-ai in the list → Click "Import"
Leave all settings as default → Click "Deploy"
Wait 30–60 seconds → Your app is LIVE! 🎉
Go to Firebase → Auth → Authorized Domains → Add your .vercel.app URL
🎉Your Deva AI app is now live at deva-ai.vercel.app — share it with the world!

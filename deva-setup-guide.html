Complete Deployment & Publishing Guide
Step-by-step guide to go from code to live app — Firebase, Vercel, Google Play Store & Apple App Store
Phase 1
Firebase Authentication Setup
~20 Minutes
Firebase free tier (Spark plan) is enough to start — supports up to 10,000 monthly active users with no cost!
1
Create Firebase Project
Go to console.firebase.google.com → Click "Add project" → Name it "deva-ai" → Disable Google Analytics (optional) → Click "Create Project"
Use your main Google account — this will be the owner of the project
2
Enable Authentication
In Firebase Console → Authentication → Get Started → Sign-in method tab → Enable these providers:
Google — Click Enable → Add your support email → Save
Email/Password — Click Enable → Save
3
Create Firestore Database
Firebase Console → Firestore Database → Create database → Select "Start in production mode" → Choose region asia-south1 (Mumbai) → Done
Then go to Rules tab and paste these security rules:
// Firestore Security Rules — Paste in Firebase Console → Firestore → Rules rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { // Users can only read/write their own document match /users/{userId} { allow read, write: if request.auth != null && request.auth.uid == userId; } // Admins can read all users match /users/{userId} { allow read: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.isAdmin == true; } // Queries log — users write their own, admin reads all match /queries/{queryId} { allow create: if request.auth != null; allow read: if get(/databases/$(database)/documents/users/$(request.auth.uid)).data.isAdmin == true; } } }
4
Get Your Firebase Config
Firebase Console → Project Settings (⚙️ gear icon) → Scroll to "Your apps" → Click "Add app" → Select Web (</>) → Register app → Copy the config object
// Your Firebase config will look like this: const firebaseConfig = { apiKey: "AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXX", authDomain: "deva-ai-12345.firebaseapp.com", projectId: "deva-ai-12345", storageBucket: "deva-ai-12345.appspot.com", messagingSenderId: "123456789012", appId: "1:123456789012:web:abcdef1234567890" };
Replace the firebaseConfig in deva-auth-firebase.html with YOUR values above
5
Add Authorized Domains
Firebase Console → Authentication → Settings → Authorized domains → Add your Vercel domain:
deva-ai.vercel.app your-custom-domain.com localhost (already added by default)
6
Make Yourself Admin
After signing up with your email, go to Firebase Console → Firestore → users collection → Find your document → Edit → Add field isAdmin = true (boolean)
The admin email admin@devaai.com is auto-set as admin. Change this to your email in deva-auth-firebase.html
Phase 2
Vercel Deployment (Web App)
~15 Minutes
Vercel Hobby plan is completely FREE — includes custom domain, SSL, CDN, and 100GB bandwidth/month!
1
Prepare Your Project Files
Create a folder called deva-ai on your computer with this structure:
deva-ai/ ├── index.html ← rename deva-auth-firebase.html ├── app.html ← rename deva-ai.html ├── admin.html ← rename deva-admin.html ├── vercel.json ← routing config (below) └── README.md ← optional
2
Create vercel.json Config
Create a file called vercel.json in your project folder:
{ "version": 2, "routes": [ { "src": "/", "dest": "/index.html" }, { "src": "/app", "dest": "/app.html" }, { "src": "/admin", "dest": "/admin.html" }, { "src": "/login", "dest": "/index.html" } ], "headers": [ { "source": "/(.*)", "headers": [ { "key": "X-Frame-Options", "value": "SAMEORIGIN" }, { "key": "X-Content-Type-Options", "value": "nosniff" }, { "key": "Referrer-Policy", "value": "strict-origin-when-cross-origin" } ] } ] }
3
Upload to GitHub
Go to github.com → Create a new repository called deva-ai (private) → Upload all your files
# If you have Git installed, run these commands in your project folder: git init git add . git commit -m "Initial Deva AI deployment" git remote add origin https://github.com/YOUR_USERNAME/deva-ai.git git push -u origin main
If you don't know Git — use GitHub Desktop app or just drag-and-drop files on github.com
4
Deploy on Vercel
1. Go to vercel.com → Sign up with GitHub
2. Click "Add New Project"
3. Import your deva-ai GitHub repository
4. Click "Deploy" — done!

Your app will be live at: deva-ai.vercel.app
Every time you push to GitHub, Vercel auto-deploys your new changes in ~30 seconds!
5
Add Custom Domain (Optional)
Vercel Dashboard → Your Project → Settings → Domains → Add your domain (e.g. devaai.in)

Phase 3
Google Play Store (Android App)
~2–3 Days
Google Play registration: One-time $25 (≈₹2,100). After that, no monthly fees!
We'll use Capacitor.js to wrap your HTML app into a native Android app — no need to rewrite in Java/Kotlin!
1
Install Required Tools
Install these on your computer (Windows/Mac/Linux):
Node.js — Download from nodejs.org (LTS version)
Android Studio — Download from developer.android.com/studio
Java JDK 17 — Included with Android Studio
# After installing Node.js, install Capacitor globally: npm install -g @capacitor/cli
2
Initialize Capacitor Project
# Create a new folder and set up Capacitor: mkdir deva-ai-mobile && cd deva-ai-mobile npm init -y npm install @capacitor/core @capacitor/cli @capacitor/android npx cap init "Deva AI" "com.sushvinailabs.devaai" --web-dir www # Create www folder and copy your HTML files: mkdir www # Copy index.html, app.html, admin.html into www/ folder
3
Create capacitor.config.json
{ "appId": "com.sushvinailabs.devaai", "appName": "Deva AI", "webDir": "www", "server": { "androidScheme": "https" }, "plugins": { "SplashScreen": { "launchShowDuration": 2000, "backgroundColor": "#05030a", "androidSplashResourceName": "splash", "showSpinner": false }, "StatusBar": { "style": "DARK", "backgroundColor": "#05030a" } } }
4
Build Android Project
npx cap add android npx cap sync android npx cap open android # This opens Android Studio automatically
In Android Studio, wait for Gradle sync to finish (first time takes 5–10 minutes)
5
Configure App Icons & Splash Screen
In Android Studio, replace these files with your Deva AI branding:
android/app/src/main/res/ ├── mipmap-hdpi/ │ └── ic_launcher.png 72×72px ├── mipmap-mdpi/ │ └── ic_launcher.png 48×48px ├── mipmap-xhdpi/ │ └── ic_launcher.png 96×96px ├── mipmap-xxhdpi/ │ └── ic_launcher.png 144×144px └── mipmap-xxxhdpi/ └── ic_launcher.png 192×192px
Use Android Asset Studio (romannurik.github.io/AndroidAssetStudio) to generate all icon sizes from one image
6
Generate Signed APK / AAB
In Android Studio: Build menu → Generate Signed Bundle/APK → Select Android App Bundle (.aab) → Create new keystore (SAVE THE KEYSTORE FILE AND PASSWORD SAFELY!) → Build Release
Never lose your keystore file! You cannot update your app without it. Store it safely on Google Drive or external drive.
7
Publish to Google Play Store
1. Go to play.google.com/console → Pay $25 registration
2. Click "Create app" → Fill details
3. Upload your .aab file to Internal Testing first
4. Fill out Store Listing:
FieldWhat to Write
App NameDeva AI — Divine Wisdom
Short DescriptionAI-powered Vedic wisdom from Bhagavad Gita & scriptures
CategoryLifestyle → Religion & Spirituality
Content RatingEveryone
ScreenshotsMin 2 phone screenshots (1080×1920px)
Feature Graphic1024×500px banner image
5. Submit for review — usually takes 3–7 days for first app
Phase 4
Apple App Store (iOS App)
~1 Week · Needs Mac
iOS development REQUIRES a Mac computer and Apple Developer account ($99/year ≈ ₹8,300/year)
If you don't have a Mac, use a Mac cloud service like MacinCloud.com (~$30/month) for the build step only
1
Prerequisites (Mac only)
Xcode — Download free from Mac App Store
Apple Developer Account — developer.apple.com ($99/year)
Node.js & Capacitor — Same as Android setup
2
Add iOS to Capacitor
npm install @capacitor/ios npx cap add ios npx cap sync ios npx cap open ios # This opens Xcode automatically
3
Configure in Xcode
In Xcode → Select your project → Signing & Capabilities tab:
Set Bundle Identifier: com.sushvinailabs.devaai
Select your Team (Apple Developer account)
Set Version: 1.0.0 and Build: 1
Add app icons in Assets.xcassets
4
Archive & Upload to App Store Connect
In Xcode: Product → Archive → Organizer window → Distribute App → App Store Connect → Upload
First time setup takes longer. After first successful upload, subsequent updates are much faster.
5
Submit on App Store Connect
Go to appstoreconnect.apple.com:
FieldWhat to Write
App NameDeva AI — Divine Wisdom
SubtitleVedic Wisdom & Dharma Guide
CategoryLifestyle (Primary), Education (Secondary)
Age Rating4+ (no objectionable content)
Screenshots6.5" iPhone screenshots (min 3)
Privacy Policy URLYour website/privacy-policy page
Review takes 1–3 days typically. First app may take up to 7 days.
Phase 5
Razorpay + In-App Payments Setup
~30 Minutes
Razorpay charges 2% per transaction. No monthly fees. Supports UPI, Cards, NetBanking, Wallets.
1
Create Razorpay Account
Go to razorpay.com → Sign up → Complete KYC
KYC needs: Aadhaar + PAN + Bank account
Business type: Select "Individual" or register a company
Approval takes 1–3 business days
2
Get API Keys
Razorpay Dashboard → Settings → API Keys → Generate Test Key first, then Live Key after KYC approval
// In deva-auth-firebase.html, replace: const RAZORPAY_KEY = 'rzp_live_XXXXXXXXXXXXXXXXX'; // For Play Store/App Store in-app purchases, use Razorpay Subscriptions API // or use Google Play Billing / Apple StoreKit instead (required for stores)
For mobile apps on Play Store / App Store, you MUST use Google Play Billing or Apple IAP for subscriptions. You cannot use Razorpay for in-app purchases in mobile apps — only for web.
